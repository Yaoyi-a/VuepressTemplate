# Install Dependence

npm install

或

npm i

# Update Dependence

npm update

# Start Dev

Open local develope environment, please visitlocalhost:8080。

npm run serve

Tips

Use Ctrl + C to stop local dev process.

# Build project

Generate the files that will eventually be deployed to the server.

npm run build
