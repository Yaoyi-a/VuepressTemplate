# 安装项目依赖

npm install

或

npm i

# 升级依赖

npm update

# 调试项目

创建开启一个本地预览，请使用localhost:8080访问。

npm run serve

提示

使用Ctrl + C终止任务本地调试进程。

# 构建项目

打包构建项目，生成最终部署到服务器上的文件

npm run build
在 GitHub 上编辑此页
上次编辑时间: 1 分钟前
