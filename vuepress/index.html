# 插件

  • active-header-links：页面滚动时自动激活侧边栏链接
  • back-to-top：添加返回顶部按钮
  • google-analytics：添加Google analytics
  • last-updated：更新时间
  • medium-zoom：图片缩放
  • nprogress：进度条
  • PWA：支持Progressive Web App
  • register-component：注册components
  • search：基于标题的搜索插件

具体用法详见：Vuepress插件

# 命令

# 安装项目依赖

npm install

或

npm i

# 升级依赖

npm update

# 调试项目

创建开启一个本地预览，请使用localhost:8080访问。

npm run serve

使用Ctrl + C终止任务本地调试进程。

# 构建项目

打包构建项目，生成最终部署到服务器上的文件

npm run build
